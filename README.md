# Primetrade.ai – Trader Performance vs Sentiment

## Methodology

Loaded both datasets, cleaned missing values, converted timestamps, merged on date, analyzed PnL, leverage, trade size, and trader behavior across Fear/Greed sentiment.

## Key Insights

1. Greed days showed higher trade activity and larger average position sizes.
2. Fear days had more defensive behavior with lower leverage.
3. High-frequency traders were more consistent than large-position traders.

## Strategy Recommendations

1. Reduce leverage and position size during Fear periods.
2. Use momentum strategies during Greed periods with strict stop-loss.
3. Prefer disciplined frequent traders over aggressive oversized traders.
